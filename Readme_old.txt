== Description ==

Make distinct and styled Boxes with a image on the left side of the box. Can be used to display headings. You can easily display upto 20 Different and Cool Boxes to display different types of Boxes. Look at the screen-shots for the type of boxes!

== Installation ==

Installing Notice Boxes is a Breeze !

To install the plugin follow these steps :

1. Download the notice-boxes.zip file to your Computer.
2. Unzip the file contents.
3. Upload the "Notice Boxes" folder to the "/wp-content/plugins/" directory
4. Activate the plugin through the 'Plugins' menu in WordPress

You can also go to the "add new" tab under plugins and then search for notice boxes. From the screen then select Notice Boxes and Install it!

That's it.
